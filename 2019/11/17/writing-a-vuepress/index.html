# 初尝vuePress

# vuePress

# 是什么？

官网给出的说法是

Vue-powered Static Site Generator(以vue为核心的静态网站生成器

# 如何开始

# 1. 先全局安装vuepress

yarn global add vuepress

或者用npm

npm -g install vuepress

# 2. 创建项目

yarn create vuepress

或者用npm

npm create vuepress

填入一些关于项目的资料后可以选择需要的模板.

# 3. 运行项目

只需要在项目文件中运行

yarn dev

或者npm

npm run dev

然后打开http://localhost:8080/就会看到项目界面了
